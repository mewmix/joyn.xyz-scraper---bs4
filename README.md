### Run

Save file as scrape.html and run the code to see the scraper download the url. Change as 
needed.


```

git clone https://github.com/mewmix/joyn.xyz-scraper---bs4

cd joyn.xyz-scraper---bs4

pip3 install -r requirements.txt

python3 media.py

python3 video.py

```
